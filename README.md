# 🏥 Detector Inteligente de Tickets Hospitalarios

Proyecto desarrollado para la asignatura *Procesamiento de Imágenes* de la *Tecnicatura Superior en Ciencia de Datos e Inteligencia Artificial (ISPC)*.

## 👥 Integrantes

* Emilce Robles
* Ian Marini

## 📌 Descripción

Este proyecto implementa un sistema de detección automática de campos en tickets hospitalarios mediante técnicas de visión por computadora y aprendizaje profundo.

Se utilizó el modelo *YOLOv8* para identificar regiones de interés dentro de documentos administrativos del ámbito de la salud.

## 📂 Dataset

Dataset utilizado:

https://universe.roboflow.com/builker/hospital-regional-sur-oriental-0boql

## 🎯 Objetivo

Desarrollar un modelo capaz de detectar automáticamente campos relevantes en tickets hospitalarios para contribuir a la automatización de procesos documentales.

## 🚀 Resultado

Se entrenó un modelo YOLOv8 capaz de detectar regiones de interés dentro de tickets hospitalarios utilizando imágenes previamente etiquetadas.
