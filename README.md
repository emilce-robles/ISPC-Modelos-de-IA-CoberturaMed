# 🏥 Detector Inteligente de Tickets Hospitalarios

Proyecto desarrollado para la asignatura *Procesamiento de Imágenes* de la *Tecnicatura Superior en Ciencia de Datos e Inteligencia Artificial (ISPC)*.

## 👥 Integrantes

* Emilce Robles
* Ian Marini

## 📌 Descripción

Este proyecto implementa un sistema de detección automática de campos en tickets hospitalarios mediante técnicas de visión por computadora y aprendizaje profundo.

Se utilizó el modelo *YOLOv8* para identificar regiones de interés dentro de documentos administrativos del ámbito de la salud.

## 📂 Dataset

Dataset utilizado:

https://universe.roboflow.com/builker/hospital-regional-sur-oriental-0boql

## 🎯 Objetivo

Desarrollar un modelo capaz de detectar automáticamente campos relevantes en tickets hospitalarios para contribuir a la automatización de procesos documentales.

📊 Etapas desarrolladas
- Validación del dataset.
- Análisis exploratorio de datos (EDA).
- Procesamiento de imágenes.
- Conversión de anotaciones a formato YOLO.
- Entrenamiento de YOLOv8.
- Evaluación mediante métricas de detección.
- Generación de predicciones.
- Desarrollo de modelo complementario Random Forest.
- Análisis de resultados y conclusiones.

🤖 Modelos implementados
YOLOv8

Modelo principal utilizado para la detección automática de campos dentro de tickets hospitalarios utilizando imágenes.

Random Forest

Modelo complementario de clasificación desarrollado como parte de la asignatura Modelos de Inteligencia Artificial para analizar características derivadas del dataset y aplicar técnicas supervisadas de Machine Learning.

## 🚀 Resultado

Se obtuvo un modelo capaz de detectar automáticamente regiones de interés dentro de tickets hospitalarios, demostrando la viabilidad de aplicar técnicas de Inteligencia Artificial para la automatización documental en entornos de salud.
